Lesson 9 Working with Margins, Padding, Alignment, and Floating

What You'll Learn in This Lesson

  • How to add margins around elements

    • Adding margins is to add extra spaces outside of an element. I can use the CSS margin property which includes margin-top, margin-right, margin-bottom, and margin-left.

      There are some shortcuts for margin property.

      • One value: The size of all the margins

      • Two value: The size of the top/bottom margins and the left/right margins (in that order)

      • The size of the top margin, the left and right margins (they are given the same value), and the bottom margin (in that order)

      • Four value: The size of the top, right, bottom, and left margins (in that order)

    • I should always set the margin of the body to be 0px because this is better for the placements of other elements.

  • How to add padding within elements

    • Adding padding is to add extra spaces within an element. I can use the CSS padding property which includes padding-top, padding-right, padding-bottom, and padding-left.

  • How to keep everything aligned

    • I should use the CSS property text-align and vertical-align to align my web content. Both text-align and vertical-align are similar, but vertical-align is to align content vertically.

      Here are some values of the vertical-align

      • top: Aligns the top of an element with the current line

      • middle: Aligns the middle of an element with the middle of its parent

      • Bottom: Aligns the bottom of an element with the current line

      • text-top: Aligns the top of an element with the top of its parent

      • baseline(default): Aligns the baseline of an element with the baseline of its parent

      • text-bottom: Aligns the bottom of an element with the bottom of its parent

    • If I want to center a box element, I should set its margin left and right to be 0.

  • How to use the float property

    • I can use the CSS float property to float my content to the left or right of a div element. Float property allows elements to be moved around in the design so that other elements can wrap around them.

    • If I want to clear float elements near by another element. I should use the CSS clear property to clear float elements at the left or the right or even both sides.

Summary

In this lesson, I learned the box model which every element have on a web page. Every element is wrapped in three layers. The first layer is the margin which is the extra space of an element. The second layer is a border, and the third layer is padding which is the extra space within an element. I can set margin and padding by the CSS property with values such as margin-top, padding-top, margin-left, and padding-left. Both properties have shortcuts that define their margin and padding. For example, one value defines all sides. Two values define top, bottom, and left, right.

If I want to align my content, I could use text-align and vertical-align. The difference between them is that text-align aligns content horizontally and vertical-align aligns content vertically. The best practice to center content is to set the left and right margins to zero instead of using a center tag. The CSS float property allows an element could be wrapped by other elements.

Lesson 10 Understanding the CSS Box Model and Positioning

What You'll Learn in This Lesson

  • How to conceptualize the CSS box model

    • Every block element has margin, border, and padding aorund them.

      • margin: is the area outside the element. It never has color; it is always transparent

      • border: extends around the element, on the outer edge of any padding. The border can be of several types, widths, and colors.

      • padding: exists around the content and inherits the background color of the content area.

      • content: is surrounded by padding.

    • The CSS outline property is also included in box model, but it doesn't take up any space.

  • How to change the box model the browser uses

    • I can use box-sizing property like width, height, float, padding, and border.

      There are two more box-sizing property

      • content-box: The width and height values are assigned to the content box only, and padding and border are added afterward. This is the default.

      • border-box: Any defined padding and border are included inside the assigned width and height values. This used to be called “quirks mode.”

  • How to position your elements

    • I need to use the CSS position property to place my content. Here are some values:

      • relative: The element is positioned relative to its current position in the document flow.

      • absolute: The element is positioned based on its container element. This property can cause element overlap, but z-index property can help me determine which element should be overlap other.

      • fixed: The element is positioned relative to the browser window.

      • static: The element is placed as it appears in the normal flow. This is the default.

      • sticky: causes the element to toggle between relative and fixed, depending on the scroll position.

    • After specify type of position of an element, I should set position for them. I can use the CSS left, right, top, and bottom property.

  • How to control the way elements stack up

    • I can use the CSS property z-index to control stack up.

    • z-index: enables you to set the order of elements with respect to how they stack on top of each other. The highest value that I assgin to z-index for an element, so that element will be prioritized others.

  • How to manage the flow of text

    • I can use the CSS property float, clear, and overflow to manage the flow of text.

    • overflow: handles overflow text, which is text that doesn’t fit within its rectangular area; Here are some values:

      • visible: display rest of the content outside of the box when rest of the content doesn’t fit in the box.

      • hidden: hidde the rest of the content that doesn’t fit the box.

      • scroll/auto: make the box is scrollable.

Summary

In this lesson, I have a full understanding of the box model and learn how to change the size of a box. If I set a width and height of 100px for a box, its width, and height currently are 100px. When I set the margin of the box to be 10px, the width and the height should be 120px because width + left margin + right margin = 100px + 10px + 10px = 120px. This is the same idea when I apply padding and border. After setting the layout, I should figure out the position of my elements. I have static, fixed, relative, absolute, and sticky types of positions. Every element is static by default. The fixed and sticky positions stay in the same position while scrolling the browser.

The most useful position types are relative and absolute position types. When I set an element to be relative, it can change its position relative to its current position and does not affect other elements. When I set an element to be absolute, it flows out of screen control but its position is relative to its nearest container. I should be aware of the downside of absolute which is the element overlap issue, but I could use z-index to easily fix this problem. If I need to control text overflow in a box, I need to use the overflow property.

Lesson 11 Using CSS to Do More with Lists, Text, and Navigation

What You'll Learn in This Lesson

  • How the CSS box model affects lists

    • I can use box-sizing property to affect margin, padding, color, and background-color of lists.

      I could even specify styles of lists like:

      • list-style-image: for placement of an image as a list-item marker

      • list-style-position: indicating where to place the list-item marker

      • list-style-type: the type of list-item marker itself

  • How to customize the list item indicator

    • I can customize the list item indicator by the CSS list-style-position property. The values are:

      • inside: put the list item indicator inside the box with the items.

      • outside: put the list item indicator outside of the box from the items.

  • How to use list items and CSS to create an image map

    • Creata a div put a list into that div. In a ul tag, I need to have a li tag. Then, I add anchor inside the li tag to define source of that area in the map. After that, I specify area of that src and use background property to show background image of that area in the style sheet.

  • How navigation lists differ from regular lists

    • The user’s mouse cursor will change to indicate that the element is clickable.

    • The area around the element changes appearance when the mouse hovers over it.

    • The content area is visually set apart from regular text.

  • How to create vertical navigation with CSS

    • A nav tag means include a navigation. To create a vertical navigation, I can simply just create a list because vertical list is the default format. However, I can use the CSS property margin, width, and float to position my list and add more features on them by link, hover, visited, and active.

      When I create a list in another list and want to style them, I should do:

      • Use nav ul and nav ul li for the first level of lists

      • Use nav ul ul and nav ul ul li for the second level of lists.

  • How to create horizontal navigation with CSS

    • I need to change the display property of ul and li to be inline to create a horizontal navigation.

Summary

In this lesson, I learned more about how to style my list and use a nav tag to navigate on my page. I can use a box model to affect my lists such as margin, border, padding, width, float, background-color, and color. If I want to customize my list item indicator, I need to use the list-style-image and the list-style-type. When I want to change the position of the list item indicator inside the box, I can use list-style-position.

Furthermore, I can use the list to create an image map. I need to add an anchor tag in a li tag and specify its URL and size. Navigation lists differ from the regular list because I have more features than a regular list. For example, when users hover over the link, the text is changed to be larger and have a different color. The vertical navigation list is defaulted by HTML, but the horizontal navigation list is not. I need to set the display of the ul and the li to be inline.
