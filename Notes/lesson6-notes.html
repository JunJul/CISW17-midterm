Lesson 12 Creating Layouts Using Modern CSS Techniques

What You'll Learn in This Lesson

  • How fixed layouts work

    • A fix layout means body of the page is set to a specific width other than percentage. That width is typically controlled by a master “wrapper” element that contains all the content.

    • The CSS property width is an exmaple if I set a div element with a width.

  • How liquid layouts work

    • A liquid layout—also called a fluid layout—is a layout in which the body of the page does not use a specified width in pixels, although it might be enclosed in a master wrapper element that uses a percentage width. The idea behind a liquid layout is that it can be perfectly usable and still retain the overall design aesthetic, even if the user has a very small or very wide screen.

  • How to create a fixed/liquid hybrid layout

    • A fixed/liquid hybrid layout contains elements of both types of layouts. For example, you could have a fluid layout that includes fixed-width content areas either within the body area or as anchor elements (such as a left-side column or as a top navigation strip). You can even create a fixed content area that acts like a frame, in which a content area remains fixed even as users scroll through the content.

  • How to create CSS flexible box layouts

    • The CSS Flexible Box module gives you a way to position elements that display on a line in a container

    • The container can alter the dimensions of the elements inside it to best fit the available free space.

    • A flexible box container, or flex container, will expand items to fill available free space or shrink them to prevent overflow in smaller spaces.

    • This is also called normal flow which means normal layout in web pages is based on block and inline elements interacting with one another

    • Inside a container, I need to set display property to be to be flex. Then, I add flex-direction property to set the flow and here are the properties: row, row-reverse, column, and column-reverse.

      Once I have the flow direction, I need to consider how the items will wrap when they come to a boundary. I can use flex-wrap property.

      After we set direction and wrap, I need to set space between each item. I should use the justify-content property.

      Finally, If I want to align and position my items vertically inside a contianer, I should use the align-item property. Moreover, if I want multiple lines in my container, I should use align-content.

      If I want to arrange the order of item, I should use the order property.

  • How to use CSS grid layouts

    • I need to define a container to be grid by the property display. Then, I should define the rows and cols with the grid-template-rows and grid-template-columns property. Finally, I can place the child elements into the grid with grid-template-areas and the grid-area property.

      fr: This refers to the free space inside the container. grid-template-columns: 15% 3fr 1fr; Have a three-column layout with a 15% left column and the remaining columns taking up three-quarters and one-quarter of the remaining available space.

  • Other concepts

    • min-width: sets the minimum width of an element, not including padding, borders, or margins.

      max-width: ensure that elements of the page with percentage widths don’t get too wide. This property lets the element expand up to a point and then stop.

Summary

In this lesson, I have learned fixed and liquid layout, and grid, flex system to organize elements on a web. The fix elements are set with a default width in pixels, so when users scroll my webpage, the fix elements stay at the same position. The liquid elements are set with a flexible width in percentage, so the width is relative to container of the liquid elements. When the size of the container is changed, the width in percentage unit can always fit the size of the container by the percentage value. If I combine fixed and liquid layout together, it means some of the elements are flexible and some of the elements are not.

The flex system is to create a flexible container that can contain elements and is responsive. When users minimize their screens, the container can resize and rearrange items in rows or cols inside the container. If I want to use a table for layout, I should consider a grid system. The grid system is the best practice for table layout because it can specify the size of the row and col, and define a gap between each cell. The grid system can design a layout for two dimensions such as a row and col, but the flex system only design for one D such as a row or col.

Lesson 13 Taking Control of Backgrounds and Borders

What You'll Learn in This Lesson

  • How to layer backgrounds

    • I can use the background properties to help me create multiple backgrounds. I should use backrgoud: url('') to insert my backgrounds in a container. Then, I need to use background-clip property to decide which part of the background should be hidden or covered by the background of the container. Finally, I need to set their background-size and background-repeat.

  • How to use gradients as backgrounds

    • I can define my backgrounds with the combination of background-image and background-color.

    • The property linear-gradient: with a comma-separated list of colors inside the parentheses. background-image: linear-gradient( #ff00d5, rgba(39, 164, 0, 0.5), blue ); I can also add color stops and color direction to stop and move my gradients.

    • The property radial-gradient: starts a color at a point. I can define the shape beofre setting the color. While defining the shape, I can give it more precious by adding position keywords such as background-image: radial-gradient(at bottom left, orange, white);

    • linear gradients fade down a line, radial gradients start at a single point and emanate outward.

  • How to use CSS properties to create “zebra-stripe” tables automatically

    • To create aebra-strip tables, I can use the nth-child(odd/even) property to set odd or even child to be a different color.

  • How to create rounded corners on elements

    • I can use the border-radius property

  • How to use images as borders

    • I can the border-image-source property to define my source of the image. Then, I need to use border-image-slice property to crop my four edges.

  • How outlines are different from borders

    • Outlines do not take up space. Outlines may be non-rectangular. Browsers and other user agents often render outlines on elements when they are in the :focus state. Unlike border, the outline is drawn outside the element's border, and may overlap other content. Also, the outline is NOT a part of the element's dimensions; the element's total width and height is not affected by the width of the outline.

Summary

In this lesson, I learned how to format my backgrounds and add multiple backgrounds on a webpage. By adding, a background on a webpage, I use the shortcut background: url(''). After this statement, I specify the position of the background image and size of the image and repetition such as background: url('') top right / 45% repeat. When I use a gradient as background, I should use linear-gradient and radial-gradient. I can also define color stops after each color to decide where should the color stop at.

To set a background color for an element, the best practice would be nth-child() because I can have better control over what color of a specific element to be. I can round the corner of an element by using the border-radius property and I can also set an image as a border by using border-image properties. Outline and border do not take any space initially; however, an outline may be not a rectangular type and a border has to be a rectangle.

Lesson 14 Using CSS Transformations and Transitions

What You'll Learn in This Lesson

  • How to transform elements by rotating, scaling, moving, and tilting

    • The trasnform: rotate() function means to rotate the element in degree.

    • The trasnform: scale() function means to change the size of the element, but it can also change the size of the element in x and y axis.

    • The transform: translate() function of the transform property moves an element from its default position on the page to the defined amount horizontally or vertically.

    • The transform: skew() means on an element to slant elements on the horizontal and vertical axes.

  • How to work with transformations in three dimensions

    • rotatex(), rotatey(), and rotate3d(x, y, z, angle): Allows you to define a point in 3D space around which to rotate the element.

    • scale3d(x, y, z) and scalez()—Resizes the element in three dimensions.

    • translate3d(x, y, z) and translatez()—Moves the element in three dimensions.

    • The best way to do this is to set the perspective property on the parent element. The value of this property determines the intensity of the effect; for example, further away makes the effect less intense than up close.

  • How to apply multiple transformations to one element

    • Here is the shortcut for trasnform porperty to define all types of the transformation. For example, transform: rotate(30deg) scale(1.2) translate(15px,0) skew(5deg);

  • How to do simple animations with the transition property

    • I need to use transition property with transform property to create simple animations. transition: property duration timing-function delay;

    • property(which property is set to have transition) duration(how long does the transition take) timing-function(speed of the transition) delay(when does the transition starts)

Summary

In this lesson, I learned basic animations in CSS with two main properties. They are the transform and the transition property. If I want to create some simple animations, I need to use the transition and the transform property. The transform property can help me to rotate, translate/position, skew/slant, and scale an element. Moreover, I can transform an element into 3D. Simply, I just need to define one more dimension when I define the transform type. For example, rotate3d(x,y,z,angle). Also, there is a shortcut that I can define all types of transformation such as transform: rotate(), translate();

However, the transform property is static and cannot interact with users, so I need to use transition. The transition property helps the transform to move fast or slow while providing dynamic transformation with users When I define transition property, I need to define transform property by using pseudo-classes. Then, I set transition: property duration timing-function delay;
