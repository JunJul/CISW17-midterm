Ch6: Working with Fonts, Text Blocks, Lists, and Tables

What You'll Learn in This Lesson

  • How to use boldface, italic, and special text formatting

    The < strong > < /strong > tag is to bold texts.

    The < em > < /em > tag is to italize texts.

    The < sup > < /sup > tag is to Superscript text.

    The < sub > < /sub > tag is to Subscript text.

    The < pre > < /pre > tag is to Monospaced text, preserving spaces and line breaks.

  • How to tweak the font

    The < font > < /font > tag allows us to tweak texts. For example, add size and color properties in the opening font tag.

    Using inline or external css to tweak fonts with font-family, font-size, and color properties.

  • How to use special characters

    Special characters are specified as either numeric codes or code names.

    When using special characters, type their nummeric codes or code names.

  • How to align text on a page

    p tag and div tag provide mobility of aligning text on a page.

    Using a style attribute called text-aligin to aligin text. We can style the aliginment by inline, internal and external css for any block element.

  • How to use the three types of HTML lists

    Ordered list: a ol tag has an indented list that has numbers or letters before each list item.

    Unordered list: a ul tag has an indented list that has a bullet or another symbol before each list item.

    Definition list: a dl tag has a list of terms and their meanings without numbers and letters before each item.

  • How to nest lists within lists

    Nested list means a list is placed in another list. Simply say, a ul tag is placed in a li tag of another ul tag.

    We can also can create a nested list with combo of ordered and unordered list.

    We can use style list-style-type to choose a bullet type.

  • How to create simple tables

    A table tag is to create a table.

    A tr tag is to create a row in a table and a td tag is to create a column in a row.

    A th tag represents header or title of each column.

    A thead tag warps your header row, a tfoot tag shows result of a colmun, and a tbody tag warps rows and columns.

  • How to control the size of tables

    We can control the size of a table by adding style width property like 50% or 25% for each row and column.

  • How to align content and span rows and columns within tables

    We can align the contents of table cells both horizontally and vertically with the text-align and vertical-align style properties.

    We can use colspand and rowspan attribute to expand or span rows and columns withn in a table.

  • How to use CSS columns

    If you have a lot of infomation to present, use CSS columns and it is better to use internal or external CSS.

    Use column-count propert y to specify how many columns we need and column-gap property to define space between each column.

    Use colspan attribute for the footer.

Summary

If I want to bold and italicize texts, I need to use strong and em tags instead of b and i tag. The reason is that the strong and the em tags are to emphasize texts and let the browser know they have special meanings. I can use special characters on a page by using numeric codes and code names. Easier, I can copy a special character from somewhere and paste it into my paragraphs. The pre, sup, and sub tags are to add style to a text to be a special character, and I can align text by using text-align.

When I want to have a list on my page, I can use list tags. We have three types of lists, ordered, unordered, and definition list. The ordered list is structured as a list by letters and numbers, the unordered list is structured list by bullet points, and the definition list is structured as a list by header and item. I can also create a simple table with a table tag. In a table, I can create rows and columns by tr and td tags. If I have a lot of information that needs to display, I should use the CSS column property to divide my content in each column.

Lesson 7: Using External and Internal Links

What You'll Learn in This Lesson

  • How to use anchor links

    The anchor tag < a href="" > < /a > is a way to link to web page. We can add images or texts between the anchor tag, so users can click texts or images to enter another page.

    href is an attribute of the anchor tag and identifies the address of the page or document to which you are linking.

    The anchor can also identify location within a page. I can add id in other tags and use the anchor tag to link it by href="#XXX"

  • How to link between pages on your own site

    The first link is an absolute link. With an absolute link, there is absolutely no doubt where the link should go because the full URL is provided—domain name included. http://www.yourdomain.com/zoo.html

    The second link is a relative-root link. It is relative to the domain you are currently browsing. /zoo.html

    In the third link, the double dot (..) is a special command that indicates the folder that contains the current folder—in other words, the parent folder. Anytime you see the double dot, just think to yourself, “Go up a level in the directory structure.” ../zoo.html

    In the same folder, I can create two sites and use an anchor tag to link the first page to the second page. Then, I can use another anchor tag in the second page to link the second page to the first page. As a result, I should use relative-root and current folder links.

  • How to link to external content

    If I need to link an external content, I need to provide the anchor tag with an absolute link, which means an external content is not in the root. It is from outside, so it needs the full URL.

  • How to link to non-Web documents such as PDF and Word documents

    Simply, I can replace URL in the anchor tag with location of a document.

    If I add download attribute in the anchor tag, it forces a browser to open download dialog.

  • How to link to an email address

    I can replcae UPR in the anchor tag with mailto:yourusername@yourdomain.com

    If I want to add title and body of a message, I can do mailto:author@somedomain.com?subject=Book Question&body= When is the next edition coming out?

  • How to use window targeting with your links

    The window targeting is a pop-up window. When I target a window, it pops up a new window.

    It is structured by < a href="" target="_blank" > < /a >

    I need to avoid using < a href="" target="newWindow" > < /a >

  • How to style your links with CSS

    I need to use pseudo-classes that describes styles for elements that apply to certain circumstances, such as various states of user interaction with that element

    a:link: Describes the style of a hyperlink that has not been visited previously.

    a:visited—Describes the style of a hyperlink that has been visited previously and is present in the browser’s memory.

    a:hover—Describes the style of a hyperlink as a user’s mouse hovers over it (and before it has been clicked).

    a:focus—Describes the style of a hyperlink as the user activates the link. This is often used on devices where there is no mouse, so clicking is not possible.

    a:active—Describes the style of a hyperlink that is in the act of being clicked but has not yet been released.

  • How to add descriptions to links

    I can add a footerlink to derscribe links that I have on my on page, so my users can enter into the footer page to see what pages are linked.

  • Best practices for web page links

    The best practices for web page links are make your links stand out, For links use only link styles, Style visited links, Use descriptive content for link text, Add padding to text links, Link images, Use icons to aid in comprehension, Avoid tiny links.

Summary

In this lesson, I learned the anchor tag links multiple pages, so users can enter another page by clicking a link on a page. Inside the anchor tag, the major attribute is href. href is an attribute to identify where users will enter and what happens to users. If I want them to enter my another page, I will type such as href="myAnotherPage.html". This is a relative-root link that stays in the same directory as the current page. If I want them to an external page, I will type the full URL of that external page.

The anchor tag can also open a document and force users' web browsers to open a donwalod dialog. I only need to add another attribute download in the anchor tag. Moreover, I want to style my links only by pseudo-classes. When a link is not visited, it has some styles. When a link is visited, its style is changed. I have hovered, visited, linked, focused, and been active. Each of them can style my link differently and make my links stand out in users' eyes.

  • How to align images

    I can horizontally align images by CSS float property.

    float: left; aligns the first image to the left and wraps text around the right side of it.

    float: left; aligns the first image to the left and wraps text around the right side of it.

    I can vertically align images by CSS vertical-align property.

    vertical-align:text-top; To line up the top of an image with the top of the tallest image or letter on the same line.

    vertical-align:text-bottom; To line up the bottom of an image with the bottom of the text

    vertical-align:middle; To line up the middle of an image with the overall vertical center of everything on the line

    vertical-align:baseline; To line up the bottom of an image with the baseline of the text.

  • How to use background images

    I should use the CSS background property. backgrround: background-color background-image background-repeat background-position

    background-image: url('imagename.gif')

    background-repeat: Specifies how the image should repeat, both horizontally and vertically. Other options are repeat (same as default), repeat-x (horizontal), repeat-y (vertical), and no-repeat (only one appearance of the graphic).

    background-position: specifies where the image should be initially placed, relative to its container. Options include top-left, top-center, top-right, center-left, center-center, center-right, bottom-left, bottom-center, bottom-right, and specific pixel and percentage placements.

  • How to use image maps

    Image maps is used to show images as navigation. To create any type of image map, you need to figure out the numeric pixel coordinates of each region within the image that you want to turn into a clickable link. These clickable links are also known as areas.

    I should use the map tag with specified a name like < map name="mapname" >. Then, I should use area tag for each region of the image. I can set attributes shape to indicate the shape of the regoin and I can set coords to indicate exact pixel coordinates for the region. For rectangles, give the x,y coordinates of the upper-left corner followed by the x,y coordinates of the lower-right corner. For circles, give the x,y center point followed by the radius in pixels. For polygons, list the x,y coordinates of all the corners in a connect-the-dots order.

    hre: Specifies the location to which the region links. You can use any address or filename that you would use in an ordinary link tag.

    alt: Enables you to provide a piece of text that describes the shape; as you learned previously, providing this text is important to users browsing with text-only browsers or screen readers.

    title: Ensures that tool tips containing the information are also visible when the user accesses the designated area.

  • How to link to or embed multimedia files

    The way link to a multimedia file is simliar to insert an image. < a href="chickadee.mov" > View a cute chickadee! < a >

  • How to use HTML5 audio and video elements

    I should use video and aduio tags to embed the videos and audios on my browser. When I use them, I need to use the attribute src="" to define my source. If I add a paragraph before a closing tag of the video, it is similar to alt in an img tag.

    Inside a video tag, we have different attributes, they are control, prload, loop, poster, and autoplay.

    controls: tells the browser to display the video with play/pause control buttons

    preload: Has three possible values: none, auto, and metadata. Use none if you do not want to buffer the file, use auto to buffer the file, and use metadata if you want to buffer only the metadata for the file.

    loop: Tells the browser to start the video from the beginning when it gets to the end.

    poster: Points to an image file that is shown when the video isn’t available, such as when it is still downloading.

    autoplay: Causes the video to start playing as soon as it is ready.
